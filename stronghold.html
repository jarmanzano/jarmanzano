Stronghold
Home Village Mineshaft Nether Fortress Stronghold

The Stronghold is a structure that naturally generates underground and is the only place where you can find an end portal. It can be located using an eye of ender which can be crafted using an ender pearl and blaze powder. In order to enter the end, you must fill up the end portal with 12 eyes of ender.
